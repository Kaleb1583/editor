# FileEditor

**FileEditor** is an HTML editor made with Monaco (Visual Studio Code). It includes various options:

### Options:
- Show alerts
- Show fullscreen button
- Show export button
- Dark mode
- Monaco font adjuster
- Reset options

### Buttons:
- Fullscreen button
- Export button
- Options button
